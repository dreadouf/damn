DAMN!

Defeasible reasoning tool for multi-agent collaboration

code About

DAMN is a defeasible reasoning platform available on the web. It allows the use of different defeasible reasoning semantics (ambiguity blocking/propagating with or without team defeat) in order to reason with incoherent or inconsistent knowledge of different agents.

build Usage

Start by logging in or creating an account! You can then create a "project". A project can have multiple knwoledge bases. Each knowledge base contains a set of facts and rules expressed using the DLGP format. Refer to the Help guide for more information.

people Multi-agent collaboration

DAMN integrates a defeasible reasoning tool with agent collaboration and visualization features. It is geared towards decision making where each agent has its own knowledge base that can be combined with other agents to detect and visualize conflicts and potentially solve them using a semantics.
